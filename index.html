Como Comprar um Plano

Siga os passos abaixo para adquirir seu plano e ter acesso aos kit bicos:

  1. Acesse o Menu: No menu principal do bot, clique no botão "Planos"
  2. Escolha seu Plano: Visualize os planos disponíveis com preços e durações
  3. Confirme a Compra: Clique no plano desejado e confirme os detalhes
  4. Realize o Pagamento: Use o PIX gerado para efetuar o pagamento
  5. Aguarde a Confirmação: Seu plano será ativado automaticamente após o pagamento
Importante: O pagamento via PIX é processado automaticamente. Mantenha o comprovante para eventuais consultas.

Como Gerar Kit Bicos

Após adquirir um plano ativo, você pode gerar quantos kit bicos desejar:

  1. Comando Direto: Digite /kitbico no chat ou use o menu
  2. Escolha a Base: Selecione entre "Meta" ou "Segurando Documento"
  3. Aguarde o Processamento: O bot irá buscar e processar os documentos
  4. Receba os Arquivos: Os kit bicos serão enviados diretamente no chat
